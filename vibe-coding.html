Le principe fondamental

Tu décris ce que tu veux → l'IA écrit le code → le code est stocké dans un repository → l'application est visible en ligne ou en local. C'est ça, le vibe coding.

🌐 Niveau 1 — Web simple

Héberger sur Internet

Le chemin le plus direct : tu codes, tu pousses, c'est en ligne. Idéal pour des pages, des outils et des prototypes.

🔗 Chaîne de liaison

Le code va du repository à l'hébergeur, automatiquement
🤖 IA ➔ 📁 Repo ➔ ⚙ Build auto ➔ 🌐 Site en ligne
Simple A

GitHub Pages

Page statique hébergée gratuitement. Chaque push met à jour le site automatiquement.

HTML/CSS/JS Gratuit
Simple B

Vercel / Netlify

Hébergement moderne avec prévisualisation par branche. Idéal pour React, Vue, etc.

React Next.js Gratuit
Moyen C

Serveur cloud

Azure, AWS ou GCP pour des applications web avec backend (Node.js, Python, etc.).

Azure AWS Backend
💻 Niveau 2 — Exécution locale

Utiliser ton ordinateur

Pas besoin d'Internet pour exécuter. Ton code tourne en local avec différentes méthodes.

🔗 Chaîne de liaison

Le code reste sur ta machine, tu l'exécutes directement
🤖 IA ➔ 📁 Dossier local ➔ 💻 Ta machine
Simple A

Fichier dans le navigateur

Double-clique sur un fichier .html : ton navigateur l'ouvre directement via file://. Zéro serveur.

HTML file:// Instantané
Simple B

Script Python

Lance python script.py dans le terminal. Traitement de données, automatisation, scraping…

Python Terminal pip install
Moyen C

Application exécutable

Un .exe ou une app Electron que tu lances comme n'importe quel logiciel sur ton bureau.

Electron .exe Bureau
🗃 Niveau 3 — Ressources externes

Connecter des ressources

Ton application a besoin de données, de stockage ou de services tiers ? La chaîne s'allonge.

🔗 Chaîne de liaison

L'application communique avec des services externes
🤖 IA ➔ 📁 Repo ➔ ▶ App ➔ 🗃 Ressources
Moyen A

Base de données relationnelle

PostgreSQL, MySQL, SQLite… pour stocker des données structurées (utilisateurs, commandes, etc.).

PostgreSQL MySQL SQLite
Moyen B

Base vectorielle

Pinecone, Chroma, Weaviate… pour la recherche sémantique et le RAG (IA + documents).

Pinecone Chroma RAG
Moyen C

Appels API

Consommer des services tiers : paiement (Stripe), email (SendGrid), IA (OpenAI, Claude)…

REST API Clés API Webhooks
Gouvernance Wavestone

Pour les données clients sensibles, privilégie l'environnement Microsoft (Azure) pour l'hébergement des bases et les appels API. Les clés API Wavestone permettent de rester dans le périmètre maîtrisé.

📌 Les niveaux de complexité

Simple

Page web statique, fichier HTML local, script Python. Aucune infrastructure à gérer.

Moyen

Base de données, API tierces, serveur cloud. Nécessite de la configuration et de la sécurité.

Avancé

App Store, application desktop, pipelines CI/CD. Chaîne complète de build et distribution.
