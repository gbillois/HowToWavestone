● Fondation

Les 3 briques essentielles

Tout projet de vibe coding repose sur ce trio. Chaque brique se connecte aux autres.

🌐 Niveau 1

Héberger sur Internet

Le chemin le plus direct : tu codes, tu pousses, c'est en ligne.

💻 Niveau 2

Exécution locale

Pas besoin d'Internet pour exécuter. Ton code tourne sur ta machine.

🗃 Niveau 3

Connecter des ressources

Ton appli a besoin de données, de stockage ou de services tiers ? La chaîne s'allonge.

Gouvernance Wavestone

Azure est recommandé pour les données clients sensibles (intégré à l'écosystème Wavestone). AWS est disponible pour certains cas d'usage, à valider avec la gouvernance IT.

🚀 Niveau 4

Publier et distribuer

La chaîne complète : du prompt IA jusqu'au store ou à l'installeur.

📈 Récapitulatif

Vue d'ensemble

Tous les chemins partent du même trio. La complexité vient de la destination.
