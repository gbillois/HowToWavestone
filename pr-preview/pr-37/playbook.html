Gouvernance Wavestone — Hébergement interne

Pour les données clients sensibles, deux options internes sont disponibles :

• Microsoft Azure recommandé — environnement intégré à l'écosystème Wavestone (Azure SQL, Azure OpenAI, Blob Storage, App Service). Privilégié pour rester dans le périmètre maîtrisé.

• Amazon Web Services (AWS) — alternative disponible pour certains scénarios (DynamoDB, S3, Lambda, Bedrock). À valider avec la gouvernance IT.

Les clés API Wavestone permettent dans les deux cas de rester dans un périmètre sécurisé et tracé.
