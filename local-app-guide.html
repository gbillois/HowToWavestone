Créer une application locale avec l’IA

D’une idée à une app qui tourne sur ta propre machine, étape par étape — rien d’hébergé en ligne. Suis le déroulé, coche chaque étape, copie les prompts.

# 100% sur ta machine # sans droits admin # ~30 min # Git · VS Code · Python · Claude Code

Astuce : déroule la page de haut en bas. Clique sur les numéros pour cocher les étapes validées et sur copier pour récupérer un prompt. Le sélecteur de langue est en haut à droite.

Le concept

Trois briques, et c’est tout

Créer une app en local repose sur trois briques qui s’enchaînent. Comprends-les une fois, tu comprends tout le reste.

Une idée → Du code → Une app qui tourne
Brique 1

Un agent IA écrit le code

À partir d’une idée en langage naturel, Claude génère le code directement sur ta machine. C’est lui qui fait le travail d’écriture.

Étape 1 — Préparer

Préparer ton poste

À faire une seule fois. Tout s’installe sans droits admin. Comptez 15 min.
0/6 étapes
1

Tester Claude + créer un GitHub

Connecte-toi à Claude (compte Wavestone) et crée un compte GitHub gratuit pour stocker et versionner ton code.
claude.ai/new ↗github.com ↗
2

Installer Git

Télécharge et installe Git (version Windows). Pas d’interface : c’est le composant qui suit les modifications de tes fichiers.
git-scm.com/install ↗
3

Installer GitHub Desktop

Le client qui synchronise tes repositories entre le web et ta machine (Documents/GitHub). Connecte ton compte GitHub. C’est lui qui permet la sauvegarde et le travail en commun.
desktop.github.com ↗
4

Installer Visual Studio Code

L’éditeur dans lequel tu travailleras. Télécharge pour Windows, puis Suivant → Installer.
code.visualstudio.com ↗
5

Installer Python pour le code avancé

Installe le Python Install Manager (recommandé via le Microsoft Store). Utile dès que Claude doit générer et exécuter du Python en local.
Microsoft Store ↗python.org ↗
6

Extension Claude Code + premier fichier

Dans VS Code : Extensions (Ctrl+Shift+X) → cherche « Claude Code » (Anthropic) → Installer. Connecte ton compte Claude, ouvre un dossier de ta machine, puis demande ton premier fichier dans le panneau Claude (icône +).
guide officiel ↗ouvrir dans VS Code ↗
Étape 2 — Construire

Créer ta première app

Ouvre un dossier dans VS Code, puis demande à Claude Code de créer un fichier. Voici deux prompts pour démarrer :

Exemple — page simple
Crée-moi une page HTML autonome nommée tetris.html : un Tetris minimaliste qui démarre sur la touche Entrée.
Exemple — script Python local
Crée-moi un script Python qui utilise en local faster-whisper pour transcrire mes fichiers audio et vidéo sur ma machine.
Human in the loop

En local, Claude demande ta permission avant d’exécuter une commande. Lis ce qu’il propose, puis valide — rien ne se lance dans ton dos.

Aller plus loin

Il n’y a pas de limite à l’imagination

Une fois la boucle maîtrisée, tu peux tout construire en local. Et l’IA reste ton meilleur allié à chaque blocage.

Bloqué ? Demande à l’IA

Toutes les questions sont permises : comment utiliser l’outil, comment ajouter tel élément, comment lancer ton script. L’IA est aussi ton tuteur.

N’oublie pas l’audit de sécurité !

Avant de partager ton app, demande à l’IA : « Fais un audit de sécurité de mon application et corrige les vulnérabilités. » Un réflexe simple qui évite bien des erreurs.

La boucle ne change pas

Idée → prompt → exécution locale → itération. Ce rythme, tu le connais désormais. À toi de jouer !
